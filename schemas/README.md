# Schemas

This directory is reserved for machine-readable schemas used by the PKIM automation surface.

Current schemas:

- `capability-manifest.schema.json`
- `run-manifest.schema.json`
- `export-manifest.schema.json`
- `profile-packet.schema.json`
- `mutation-result.schema.json`

If an artifact becomes important enough to automate against, give it a schema.
